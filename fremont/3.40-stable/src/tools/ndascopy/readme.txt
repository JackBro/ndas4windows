NDAS Copy is an application to copy data from local drive to NDAS device sector by sector.

First you must make sure that NDAS service is not alive as NDAS Copy use LPX dgram incomming port.
Second, you must make sure that the target NDAS service is not mounted.

Select local HDD or partition as source and select 1 or more NDAS as target.
Click start button and wait until process complete.
