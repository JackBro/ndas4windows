Release Note for NDAS Software 3.11.1325

What's New

NDAS Software Version 3.11 is a new software release with following improvements and bug fixes

  • New icons and UI for NDAS management tool and bind management tool.
  • Improved RAID1 mechanism.
    • Users of mirroring or RAID1 made by prior version of software are required to use "Migrate" command from NDAS bind management tool.
    • New RAID1 disks cannot be used by older software. Aggregation or striped (RAID0) disk can still be used by older software.
    • Supports write-sharing of RAID1 disks.
    • Supports spare disk for RAID 1. Spare disk is used as RAID1 member when one of RAID1 member disk is missing.
    • Automatic RAID1 recovery is added. Manual RAID1 recovery is now obsolete and removed from bind management tool.
  • Added "Export" function
    • Export function is used for transferring NDAS registration information from one PC to another.
    • You can access this function from NDAS device's property menu and registration wizard.
    • NDAS export files can be used on another host just by double-clicking the export file.
  • Restricted usage of NDAS device when it is needed.
    • Operations that require a whole disk access such as scandisk, changing partition and formatting are not available when multiple hosts are accessing the NDAS devices to prevent loss of data.
    • While one host is using the NDAS device exclusively, another host cannot access the NDAS device.
  • Notable bug fixes
    • Fixed a problem that volume name changed by another host is not reflected.
    • Fixed a problem that write-sharing did not work in some systems when they rebooted with NDAS device mounted.
    • Fixed a problem that write-sharing did not work when a user changes properties of network interface.
    • Fixed a problem that a crash occurs when some systems with multiple CPUs run with high network and CPU loads.
    • Fixed a problem that system crash (blue screen) occurs when jumbo frame is enabled.
    • Fixed a problem that the performance is low with network configurations that gigabit network and 100Mbps network are mixed, and with the situation that multiple NDAS devices are used at the same time.
    • Fixed a problem that NDAS management tool sometimes stalls when NDAS device is a broken unit.
  • Miscellaneous
    • One-step registration is removed to reduce confusion. Registration wizard is available only when registering new NDAS devices.

Release Note for NDAS Software 3.10.1230

What's New

NDAS Software Version 3.10 is a new software release containing updated management framework for NDAS devices, including:

  • Improved User Interfaces
    • Added more hardware properties
    • Added information about NDAS hosts using the device
    • Added options dialogs
    • Added Suspend/Hibernation support
    • Unicode support for better multilingual support
    • Added a feature to deactivate the unused NDAS devices
  • Improved Setup
    • More robust support for device driver installations
    • Single MSI for multiple languages (MUI version only)
    • Upgrade support
  • Software Update
    • Checks for the new software release from the setup and the management software
  • Bind system
    • Supports new bind system of RAID 0, RAID 1. The aggregation system is supported also
    • An user can recover the defected RAID 1 system using the bind management tool
    • New bind system does not support a mirroring system anymore, but provides a function to migrate mirroring system to a RAID 1 system without losing any data
  • Miscellaneous
    • SNMP components are removed from the software
    • Smaller footprint of the software (around 900KB for the whole software without Windows Installer Redistributable)
    • New Command-line interface to manage the NDAS devices
  • NDAS SDK (OEM partners only)
    • New NDAS device management APIs are provided for customized applications
