NDASComm API Library 0.2

This library is provided to support manufacturing processes and 
for diagnostic purpose. Do not redistribute this library.

Folder description

readme.txt : This file
doc : Various type of ndascomm documents. All files contains same contents.
lib : NDASComm API DLL library(ndascomm.dll) and DLL import library(ndascomm_dll.lib)
inc : Header files to use of NDASComm API library.
src\samples : Sample reference program source files.
